## 写在前面

> 这个项目是基于技术派的二次开发，基于SpringBoot3、MyBatis-Plus、MySQL、Redis、ElasticSearch、MongoDB、Docker、RabbitMQ、Nginx、Let's Encrypt 等技术栈实现，实现了前后端分离。
